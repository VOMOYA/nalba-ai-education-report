# 海の命 学習アプリ — プロジェクト概要

## 目的

小学6年生国語「海の命」（立松和平 著）の教材として、太一の心の葛藤を体験的に学べるインタラクティブアプリ。

## 対象

- 対象学年：小学6年生
- 教科：国語
- 単元：「海の命」（立松和平）

## 画面構成

```
┌───────────────────────────────┐
│   🌊 海の命 — 太一の選択      │ ← ヘッダー
├───────────────────────────────┤
│  「吹き出し：本文の重要な一節」│ ← スライダーに連動
│                               │
│     🐟 巨大なクエのSVG  🤔   │ ← 太一の表情が変化
│                               │
│ [打つ] ◀━━━━●━━━━▶ [打たない] │ ← スライダー
│                               │
│ 📖 語彙注釈（タップで表示）   │
└───────────────────────────────┘
```

## 機能仕様

| 機能 | 内容 |
|------|------|
| スライダー | 「打つ（復讐・手柄）」←→「打たない（共生・命）」の5段階 |
| 太一の表情 | スライダー位置に応じてemoji変化（😠→🤔→🌊） |
| 吹き出し | 各段階で本文・じいさの言葉など重要な一節を表示 |
| 葛藤メーター | 中間（50付近）で最大値になる視覚的バー |
| 語彙注釈 | 「海の主」「葛藤」「共生」など6語をタップで意味表示 |

## スライダー段階と表示テキスト

| 値 | 表情 | 心の状態 | 表示テキスト |
|----|------|---------|-------------|
| 0-15 | 😠 | 怒り・復讐心 | 「おとうを死なせた主を打ち取る！」 |
| 16-30 | 😤 | 決意・焦り | 「村一番の漁師になって仇を討つ…」 |
| 31-55 | 🤔 | 迷い・葛藤（デフォルト） | 「海の命を奪ってはならない。」（与吉じいさ） |
| 56-75 | 😟 | 躊躇・揺らぎ | 「打てば一人前…でも」 |
| 76-100 | 🌊 | 共生・感謝 | 「このクエは海の命だから…打てなかった」 |

## 技術仕様

- **技術スタック**：HTML / CSS / JavaScript（バニラ、ライブラリなし）
- **動作環境**：ブラウザで開くだけ（インストール不要）
- **イラスト**：SVGで描画（外部画像不要）
