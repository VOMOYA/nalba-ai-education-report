### 「探求」と「探究」— ニュアンスと使い分け

|  | 探求（たんきゅう） | 探究（たんきゅう） |
|---|---|---|
| **漢字の意味** | 求＝「求める・さがす」<br>→ 広く“求める”動きに重心 | 究＝「きわめる・つきつめる」<br>→ 深く“究める”姿勢に重心 |
| **主なニュアンス** | **追い求める・探し求める**<br>（目的そのものを見いだすまでの過程を強調） | **掘り下げて究明する・研究する**<br>（事象を徹底的に解明するプロセスを強調） |
| **よく使う場面・語句** | - 真理の探求<br>- 幸福の探求<br>- 探求心・探求者 | - 探究学習<br>- 科学的探究<br>- 探究活動・課題探究 |
| **教育文脈での位置づけ** | ◎「主体的な問いを**探し**に行く」段階を示すことが多い | ◎問いを立てた後、**深く調べ・検証**する段階を示すことが多い |
| **例文** | - 彼は人生の意味を**探求**している。<br>- 芸術表現の可能性を**探求**する。 | - 生徒たちは地域課題を**探究**し、解決策を提案した。<br>- DNAの仕組みを**探究**する研究が進む。 |

1. **広く求めるか、深く究めるか**
