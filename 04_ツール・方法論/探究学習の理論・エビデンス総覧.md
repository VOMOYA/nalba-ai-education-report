## 理論的基盤：5人の巨人

探究学習の思想は20世紀初頭から積み重なってきた構成主義の系譜に立つ。

```mermaid
graph TD
  D["デューイ（1910s–）\n経験と学習・探究の哲学"]
  P["ピアジェ（1920s–）\n認知構成主義・能動的学習"]
  V["ヴィゴツキー（1930s–）\n社会文化理論・ZPD・足場かけ"]
  C["構成主義\nConstructivism"]
  B["ブルーナー（1960s–）\n発見学習・スパイラルカリキュラム"]
  S["シュワブ（1960s–）\n探究としての科学・レベル分類"]
  IBL["現代の探究学習\nIBL / PBL / PjBL"]

  D --> C
  P --> C
  V --> C
  C --> B
  C --> S
  B --> IBL
  S --> IBL
```

### ジョン・デューイ（John Dewey, 1859–1952）

**主著：**
- *How We Think*（1910）— 反省的思考（reflective thinking）の5段階を提唱
- *Experience and Education*（1938）— 経験主義教育の理論的集大成

**核心メッセージ：**
「学習とは経験の再構成である」。科学は記憶すべき事実ではなく、思考・探究の**方法**として教えられるべき。

> *"Learning by doing"*（実践による学習）

---

### ジャン・ピアジェ（Jean Piaget, 1896–1980）

**核心概念：**

```mermaid
graph TD
  A["既存のスキーマ"] --> B["新しい経験と出会う"]
  B --> C{"うまく当てはまる?"}
  C -->|"Yes"| D["同化（assimilation）\n新情報を既存スキーマに取り込む"]
  C -->|"No → 認知的葛藤！"| E["調節（accommodation）\nスキーマ自体を修正・再構成する"]
  D --> F["均衡化（equilibration）"]
  E --> F
  F --> G["新しい認知構造が生まれる ＝ 学習！"]
```

探究活動が「認知的葛藤」を生み、調節を促すことで深い学習が起きる。

---

### レフ・ヴィゴツキー（Lev Vygotsky, 1896–1934）

**最近接発達領域（ZPD）：**

```mermaid
graph TD
  C["① 一人でできること\n（現在の発達水準）"]
  B["② ZPD 最近接発達領域\n支援があればできること\n← ここに足場かけを投入！"]
  A["③ 今はできないこと\n（遠い将来の目標）"]

  C -->|"足場かけ（scaffolding）"| B
  B -->|"内化・発達"| C
  B -->|"次の目標へ"| A

  style B fill:#FFD700,color:#000
```

IBL における教師・仲間・ツールによる**足場かけ（scaffolding）**の理論的根拠。
（"scaffolding" の概念化：Wood, Bruner & Ross, 1976）

---

### ジェローム・ブルーナー（Jerome Bruner, 1915–2016）

**発見学習とスパイラルカリキュラム：**

```mermaid
graph BT
  S["小学校\n基礎的な概念との最初の出会い"]
  M["中学校\n概念の拡張・応用"]
  H["高校\nより抽象的・複雑な探究"]

  S -->|"同じ概念を螺旋的に深化"| M
  M -->|"同じ概念を螺旋的に深化"| H
```

同じ概念を発達段階に合わせて**螺旋的に深化**させる設計原理。*The Process of Education*（1960）で提唱。

---

### ジョセフ・シュワブ（Joseph Schwab, 1909–1988）

「**探究としての科学（science as inquiry）**」を体系化。探究レベルの分類（後述）の先駆者。1960年代 BSCS（生物科学カリキュラム研究）プロジェクトに直接反映。

## 探究のレベル分類

**Banchi, H., & Bell, R. (2008). "The Many Levels of Inquiry." *Science and Children*, 46(2), 26–29.**

```mermaid
graph LR
  L1["レベル1 確認型\nConfirmation\n問い:教師\n手順:教師\n結論:既知\n→ 手順練習・概念確認"]
  L2["レベル2 構造型\nStructured\n問い:教師\n手順:教師\n結論:生徒\n→ 証拠から説明構築"]
  L3["レベル3 誘導型\nGuided\n問い:教師\n手順:生徒\n結論:生徒\n→ 手順設計の自律性"]
  L4["レベル4 開放型\nOpen\n問い:生徒\n手順:生徒\n結論:生徒\n→ 完全な探究の自律"]

  L1 -->|"生徒の自律性↑"| L2
  L2 -->|"生徒の自律性↑"| L3
  L3 -->|"生徒の自律性↑"| L4

  style L1 fill:#cce5ff
  style L2 fill:#99ccff
  style L3 fill:#66b2ff
  style L4 fill:#3399ff,color:#fff
```

> **実践上の推奨：** 低レベルから始め、生徒の習熟に合わせて段階的に開放型へ移行する。日本の現行実践の多くは「誘導型（レベル3）」に相当。

## 主要モデル：5EモデルとPOEモデル

### 5E モデル（Bybee et al., BSCS, 1987）

最も広く普及した授業設計モデル。Rodger Bybee が開発。

```mermaid
graph TD
  E1["① Engagement 導入・関与\n先行知識の活性化・動機づけ・問いの生成"]
  E2["② Exploration 探索\n実験・観察・データ収集\n※教師は最小限の説明に留める"]
  E3["③ Explanation 説明・概念化\n生徒が証拠に基づき説明を構築\n教師が概念を形式化・補足"]
  E4["④ Elaboration 発展・応用\n新概念を新しい文脈・問題に適用"]
  E5["⑤ Evaluation 評価\n形成的・総括的評価\n※各フェーズを通じて継続的に実施"]

  E1 --> E2 --> E3 --> E4 --> E5

  style E1 fill:#f9f,color:#000
  style E2 fill:#fcf,color:#000
  style E3 fill:#cff,color:#000
  style E4 fill:#cfc,color:#000
  style E5 fill:#ffc,color:#000
```

**エビデンス：** Polanin et al. (2024) のメタ分析で効果量 **g = 0.82**（通常授業と比較）。

---

### POE モデル（White & Gunstone, 1992）

**Predict（予測）→ Observe（観察）→ Explain（説明）**

```mermaid
graph TD
  P["Predict 予測\n「何が起きると思う？」\n先行知識・素朴概念（misconception）を顕在化"]
  O["Observe 観察\n実際に見て・体験して確認する"]
  E["Explain 説明\n予測と結果のズレを説明する\n← 認知的葛藤 → 概念変容が起きる！"]

  P --> O --> E
  E -->|"新たな問いが生まれる"| P
```

**特徴：** 子どもの「思い込み（misconception）」を授業の出発点にできる。理科の実験・示範（demonstration）に特に有効。

## 日本の探究学習：総合的な探究の時間

### 制度的経緯

```mermaid
timeline
  title 日本の探究学習の制度的変遷
  1998 : 「総合的な学習の時間」を学習指導要領に導入
  2002 : 全国実施開始
  2018 : 高校学習指導要領改訂 / 「総合的な探究の時間」に名称変更・強化
  2022 : 高校での完全実施開始
  2025 : 文科省「現状・課題と検討事項」公表
```

### 「学習の時間」と「探究の時間」の相違

| 観点 | 総合的な**学習**の時間（小・中） | 総合的な**探究**の時間（高） |
|------|-------------------------------|--------------------------|
| 焦点 | 課題解決能力・生き方 | 自己の在り方生き方と一体的な課題探究 |
| 問いの出どころ | 与えられた文脈も可 | 生徒自身が実社会から問いを見出す |
| 深化の方向 | 問題解決重視 | 問い発見→探究サイクルの深化 |
| 自律性 | 段階的 | より高い自律性を期待 |

## 批判・限界・課題

### 最大の批判：認知負荷理論からの攻撃

**Kirschner, Sweller & Clark (2006). "Why Minimal Guidance During Instruction Does Not Work."**
*Educational Psychologist*, 41(2), 75–86.

```mermaid
graph TD
  A["ワーキングメモリは容量制限がある"]
  B["探究中の問題解決でWMを使い切る"]
  C["長期記憶への定着が起きない"]
  D["初心者には直接教授法の方が効率的"]
  E["専門家の誤謬（expert blind spot）\nエキスパートの思考プロセスを初心者に適用"]

  A --> B --> C --> D --> E

  style E fill:#ffcccc
```

**反論（Hmelo-Silver et al., 2007）：**
> IBL・PBL は「最小限ガイダンス」ではなく、**高度に構造化され足場かけを多用したアプローチ**である。Kirschner らの批判は不当な一括化。

---

### その他の主要な課題

| 課題の種類 | 内容 |
|-----------|------|
| **研究の質** | 小規模・単一学校・短期間の研究が多い。「探究学習」の定義が研究間で不統一 |
| **公平性** | 低SES・ELL・特別支援生徒は探究に必要な背景知識・メタ認知スキルを持ちにくい（Tanchuk, 2020） |
| **実施コスト** | 教師の高度なファシリテーション能力が必要。時間的コストが大きい |
| **評価の困難** | プロセス評価のツール・リテラシー不足 |
| **学習段階依存** | 初等教育段階では効果量が小さくなる傾向 |
| **領域依存** | 理科・STEM 以外の領域では研究・エビデンスが薄い |
